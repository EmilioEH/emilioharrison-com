description: Check for new user feedback and bug reports
---

1. Run the feedback sync script:

   ```bash
   npm run sync:feedback
   ```

   // turbo

2. Read the active feedback reports:

   ```bash
   cat docs/feedback/active-reports.md
   ```

3. If there are new reports, read the `docs/feedback/active-reports.md` file content to understand the issues.

4. If you are fixing a bug reported in the feedback, reference the Report ID in your task.
